You are a senior full-stack engineer. 0) Product Name

1) Tech Stack Requirements

Build with the following stack (do not change unless needed for Replit limitations):

Frontend: Next.js (App Router) + TypeScript

UI: TailwindCSS + shadcn/ui (Radix) components

State/Data: React Query (TanStack Query) or Server Actions where appropriate

Backend: Next.js API Routes (or server actions) + Node runtime

Auth: NextAuth.js (Auth.js) with Google OAuth (Google SSO) + session management

F) Table View (Monday)

text

number

date

multi select

checkbox

sorting and filtering

G) Docs (Notion-like)

I) Notifications
